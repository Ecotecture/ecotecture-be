## Backend repository for Ecotecture, Hackfest 2024 project.

--- 

### Deskripsi Aplikasi
Ecotecture adalah sebuah aplikasi yang bertujuan memudahkan pemilik bangunan dan sektor real estate untuk mengurangi emisi karbon bangunan.
